# Jing

> 🎯 **打造你的专属 AI 伙伴** — 让虚拟角色真正"活"起来

Jing 是一个下一代智能虚拟人后端框架，基于 [pipecat](https://github.com/pipecat-ai/pipecat) 构建，让 AI 角色拥有真实的情绪、持久的记忆、独特的声音和丰富的动作表现。

---

## ✨ 为什么选择 Jing？

### 🎭 情绪 AI Agent — 真正的情感交互

不再是冰冷的问答机器。Jing 基于 **PAD 情绪模型** 构建情绪系统，AI 会根据对话内容产生真实的情绪变化：

- 😊 开心时语气轻快，动作活跃
- 😢 伤心时语调低沉，动作迟缓
- 😠 生气时言辞犀利，情绪外放
- 😌 平静时温和自然，举止从容

情绪会影响语音语调、动作选择、回复风格 —— 就像真实的人类一样。

### 🧠 长期记忆系统 — 越聊越懂你

第一次对话，她会记住你的名字；第十次对话，她已经了解你的喜好。

- **语义记忆**：记住你告诉 TA 的所有事实 —— 姓名、生日、爱好、故事
- **对话记忆**：自动提取关键信息并持久化存储
- **智能检索**：基于向量相似度，精准回忆相关经历
- **记忆压缩**：长对话自动总结，保留精华不丢失

### 🎙️ 声音克隆 — 只需几秒，打造专属声线

提供几秒钟的音频样本，AI 就能用这个声音与你对话：

- 支持 CosyVoice、Cartesia 等多种 TTS 引擎
- 情绪感知语音合成，让声音带着情绪说话
- 低延迟流式输出，对话流畅自然

### 🎬 自由动作系统 — 你的角色你做主

内置 MMD 动作支持，让你的角色动起来：

- **VMD 动作导入**：上传你喜欢的 VMD 动作文件
- **AI 智能标签**：自动分析动作特征并生成搜索标签
- **场景化触发**：待机、思考、空闲等不同场景自动切换
- **视频预览**：上传动作预览视频，方便管理

### 🎨 高度可定制 — 从性格到形象

每一个角色都可以完全自定义：

```
config/characters/{your_character}/
├── character.json    # 角色基础配置
├── personality.md    # 性格描述（AI 会学习）
├── model/            # MMD 模型文件
└── voice.mp3         # 声音克隆样本
```

告诉 TA 的性格、给 TA 换个形象、让 TA 用你的声音说话 —— 一切由你定义。

## 🚀 快速开始

```bash
# 克隆项目
git clone https://github.com/arthur-9527/jing.git
cd jing

# 安装依赖
pip install -r requirements.txt

# 配置环境变量
cp .env.example .env
# 编辑 .env 填入你的 API Keys

# 启动服务
python main.py
```

## 🔧 核心技术栈

| 模块 | 技术选型 |
|------|----------|
| **核心框架** | [pipecat](https://github.com/pipecat-ai/pipecat) - 实时语音 AI 框架 |
| **后端框架** | FastAPI + Uvicorn |
| **AI 对话** | 支持 OpenAI、Cerebras、LiteLLM 等多种 LLM |
| **语音识别** | 阿里云 Qwen ASR / Deepgram |
| **语音合成** | CosyVoice / Cartesia |
| **向量数据库** | PostgreSQL + pgvector |
| **缓存层** | Redis |
| **动作格式** | VMD (MikuMikuDance) |
